# Vignan Timetable Management System

Full-stack timetable management system with React frontend and Node.js backend.

## Quick Start

1. **Setup**: `npm run setup`
2. **Start**: `npm start`
3. **Access**: 
   - Frontend: http://localhost:3000
   - Backend API: http://localhost:5000

## Architecture

- **Frontend**: React with Tailwind CSS
- **Backend**: Node.js + Express + MongoDB
- **Database**: MongoDB with Mongoose
- **Integration**: REST API with fallback to local data

## Features

- Section timetable management
- Faculty schedule tracking  
- Room availability monitoring
- Real-time data synchronization
- Excel import/export
- Responsive design with dark mode

## Development

- **Frontend only**: `npm run frontend`
- **Backend only**: `npm run backend`
- **Build**: `npm run build`# msd-project
